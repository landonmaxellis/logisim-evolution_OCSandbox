Detailed Description

When the user clicks a counter using the Poke Tool, a CounterPoker object is created, and that object will handle all user events. Note that CounterPoker is a class specific to GrayCounter, and that it must be a subclass of InstancePoker in the com.cburch.logisim.instance package.
