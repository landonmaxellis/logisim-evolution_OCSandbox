Detailed Description

Parameters
painter= the instance painter you have to use to create Graphics (Graphics g = painter.getGraphics())
x= if drawgates is false or not used, the component's left side; if drawgates is true it gets the component's width, subtracts 20 (for GND or Vcc) and divides for the number of outputs for each side, you'll get the x coordinate of the leftmost input -10 before the last output
y= the component's upper side
height= the component's height
up= true if drawgates is true when drawing the gates in the upper side (introduced this because can't draw upside down so you have to write what to draw if down and up)
