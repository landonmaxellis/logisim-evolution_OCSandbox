Detailed Description

Provides a unified interface for displaying various types of dialogs.

This class wraps the functionality of JOptionPane and provides additional logging capabilities when the application runs in a non-GUI (tty) mode. In GUI mode, dialogs are shown as they would be using the JOptionPane, while in non-GUI mode, relevant messages are logged instead of displaying a dialog.
