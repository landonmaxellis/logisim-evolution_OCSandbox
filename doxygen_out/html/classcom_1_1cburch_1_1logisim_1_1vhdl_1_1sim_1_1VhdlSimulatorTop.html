Detailed Description

VHDL simulator allows Logisim to simulate the behavior of VHDL architectures. It delegate this task to Questasim. Communication between Logisim and Questasim is done by a TCL socket (TclBinder). Path to Questasim has to be specified in Logisim's preferences.
