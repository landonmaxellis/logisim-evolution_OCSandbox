◆ SQOP_DFF

final int com.cburch.logisim.std.hdl.DenseLogicCircuit.SQOP_DFF = 0
static

D-flipflop[D, C, Q, X]: When cell[C] goes to high, set cell[Q] to cell[D]. Uses seqData[X] to detect this change. Notably, DFF scripts may reuse resources to build more complex flip-flops that all manipulate the same bit of memory. The flip-flop is coerced at all times to be either high or low to prevent errors spreading.

◆ SQOP_LATCH

final int com.cburch.logisim.std.hdl.DenseLogicCircuit.SQOP_LATCH = 1
static

D-latch[D, E, Q]: Whenever cell[E] is high, set cell[Q] to cell[D]. When reading from cell[D], it is coerced to be either high or low.
