◆ propagate()

void com.cburch.logisim.std.hdl.VhdlEntityComponent.propagate ( InstanceState  state)
inline

Propagate signals through the VHDL component. Logisim doesn't have a VHDL simulation tool. So we need to use an external tool. We send signals to Questasim/Modelsim through a socket and a tcl binder. Then, a simulation step is done and the tcl server sends the output signals back to Logisim. Then we can set the VHDL component output properly.

This can be done only if Logisim could connect to the tcl server (socket). This is done in Simulation.java.
