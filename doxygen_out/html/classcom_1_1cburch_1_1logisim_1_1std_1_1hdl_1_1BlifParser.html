Detailed Description

Parses a BLIF file. The BLIF file must contain a single circuit made up of a fixed set of supported subcircuits. The objective here is to provide a Yosys-compatible target. This can be compiled with something like: write_blif -icells -conn test1.blif Some effort has been made to support other configurations of Yosys, but icells must be used.
