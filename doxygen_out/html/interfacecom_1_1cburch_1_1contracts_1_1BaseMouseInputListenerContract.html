Detailed Description

Dummy implementation of javax.swing.event.MouseInputListener interface. The main purpose of this interface is to provide default (empty) implementation of interface methods as, unfortunately JDKs interfaces do not come with default implementation even they easily could. Implementing this interface instead of the parent one allows skipping the need of implementing all, even unneeded, methods. That's saves some efforts and reduces overall LOC.
