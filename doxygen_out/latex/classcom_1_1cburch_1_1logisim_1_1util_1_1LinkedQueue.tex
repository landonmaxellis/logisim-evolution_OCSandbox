\doxysubsection{Detailed Description}
A simple linked-\/list priority queue implementation, using values that extend type QNode. This supports (approximately) a subset of the java.\+util.\+Priority\+Queue API, but only enough to support Propagator. Objects in the queue must be subclasses of QNode.
