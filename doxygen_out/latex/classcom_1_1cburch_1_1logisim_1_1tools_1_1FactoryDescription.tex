\doxysubsection{Detailed Description}
This class allows an object to be created holding all the information essential to showing a Component\+Factory in the explorer window, but without actually loading the Component\+Factory unless a program genuinely gets around to needing to use it. Note that for this to work, the Component\+Factory class must be public, and it must include a public no-\/arguments constructor.
