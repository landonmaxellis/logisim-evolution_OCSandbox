\doxysubsection{Detailed Description}
VHDL simulator allows Logisim to simulate the behavior of VHDL architectures. It delegate this task to Questasim. Communication between Logisim and Questasim is done by a TCL socket (Tcl\+Binder). Path to Questasim has to be specified in Logisim\textquotesingle{}s preferences.
