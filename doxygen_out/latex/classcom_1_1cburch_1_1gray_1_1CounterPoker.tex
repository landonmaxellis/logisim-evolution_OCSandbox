\doxysubsection{Detailed Description}
When the user clicks a counter using the Poke Tool, a Counter\+Poker object is created, and that object will handle all user events. Note that Counter\+Poker is a class specific to Gray\+Counter, and that it must be a subclass of Instance\+Poker in the com.\+cburch.\+logisim.\+instance package.
