\doxysubsection{Detailed Description}
This file is adopted from Rob Camick\textquotesingle{}s demo described here\+: \href{https://tips4java.wordpress.com/2009/05/23/text-component-line-number/}{\texttt{ https\+://tips4java.\+wordpress.\+com/2009/05/23/text-\/component-\/line-\/number/}}

This class will display line numbers for a related text component. The text component must use the same line height for each line. Text\+Line\+Number supports wrapped lines and will highlight the line number of the current line in the text component.

This class was designed to be used as a component added to the row header of a JScroll\+Pane.
