\doxysubsection{Detailed Description}
Hdl\+Model is a base interface for objects that contain mutable text representing code written in an HDL. Listeners may be attached to appropriately handle updates. Implementors should typically use Hdl\+Content. It should not be confused with the other Hdl\+Model, located in the logisim.\+vhdl.\+base package.
