\doxysubsection{Detailed Description}
A linked-\/list priority queue of queues implementation, using values that extend type QNode. This supports (approximately) a subset of the java.\+util.\+Priority\+Queue API, but only enough to support Propagator. Objects in the queue must be subclasses of QNode.
